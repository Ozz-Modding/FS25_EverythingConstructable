# Everything Constructable

A Farming Simulator 25 mod that transforms instant building placement into a multi-phase construction process. Furthermore, it allows you to supply to materials towards the cost of any placeable.

[![ko-fi](https://ko-fi.com/img/githubbutton_sm.svg)](https://ko-fi.com/M4M41K5NPV)

## How It Works

When you place a building:

1. Terrain is flattened as normal
2. A construction fence is erected around the site
3. A construction project is created spanning multiple months (based on building cost)
4. You pay only a deposit upfront

## Construction Modes

**Automatic** — Monthly labour charges are deducted and the build progresses each month until complete.

**Wait for Resources** — Deliver materials (wood beams, planks, cement, bricks, roof plates, etc.) to the construction site to advance each phase. No monthly charges in this mode.

Switch between modes at any time by approaching the site on foot and opening the construction dialog.

## Settings

- **Construction Enabled** — Enable or disable the construction system
- **Labour Fraction** — Percentage of building cost paid as monthly labour (25%–75%), the remainder is the material budget
- **Material Supply Bonus** — Discount on required materials when using resource delivery mode (0%–10%)

## Included Placeables

The mod includes a set of free construction fences (7 styles) and construction signs (2 styles) that can be placed independently via the fences and decoration menu.
